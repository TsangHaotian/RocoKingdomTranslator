# 洛克王国语翻译器

一个用于翻译英文与洛克王国语之间转换的在线工具。

## 功能

- **字母对照表**：显示 26 个英文字母与洛克王国语的映射关系
- **英文 → 洛克王国语**：输入英文，自动翻译为洛克王国语
- **洛克王国语 → 英文**：
  - 使用虚拟键盘输入洛克王国语字符
  - 点击"绑定物理键盘"按钮，可直接使用键盘输入
  - 支持实时翻译
- **复制功能**：一键复制翻译结果，方便在其他地方粘贴使用

## 使用方法

直接双击打开 `index.html` 即可使用。
